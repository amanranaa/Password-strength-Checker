# Password-strength-Checker
This web application is a Password Strength Checker built using HTML, CSS, and JavaScript. The user interface invites the user to enter a password, with a show/hide toggle to view or mask the characters.

## About the Project  
This project provides a clean and modern UI for checking password strength. As a user types in a password, the app evaluates its length, character variety (uppercase, lowercase, digits, special characters) and gives immediate visual feedback: a coloured meter bar, text rating (Weak/Medium/Strong), and suggestions on how to improve it.

## Features  
- Input box with show/hide toggle for the password  
- Real-time strength assessment as the user types  
- Strength meter bar that changes colour (e.g., red ‚Üí orange ‚Üí green) based on strength  
- User-friendly, responsive design that works on desktop and mobile
